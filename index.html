Google Sheets API Key

A read-only key used to display your cellar. The app adds wines via a secure backend server.

Get from Google Cloud Console → Create Credentials → API key. Enable the Sheets API.

Sheet ID

Pre-filled with your cellar sheet. Edit if needed.

⚠️ Required sharing setting

In Google Sheets → Share → Change to Anyone with the link → Viewer. Without this the API key returns a 403 error.
